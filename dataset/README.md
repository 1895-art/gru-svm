2013 Dataset from Kyoto University
===

The [traffic data](http://www.takakura.com/Kyoto_data/ext_old_data201704/) to be used for the training and evaluation of the proposed GRU-SVM model is from the honeypot systems of Kyoto University. You may read the document describing the data format [here](http://www.takakura.com/Kyoto_data/BenchmarkData-Description-v5.pdf).
